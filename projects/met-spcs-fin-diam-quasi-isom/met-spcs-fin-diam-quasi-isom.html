All metric spaces of finite diameter are quasi-isometric

Ifan Howells-Baines

October 2024

1 Introduction

This is a short document explaining how all metric spaces of finite diameter are quasi-isometric. We will give all the required definitions before proving the result. This work is supporting the writing of my project in geometric group theory and is intended to cement my understanding of the concepts involved.

2 Definitions

In this section we will give the definitions needed to understand the result we are trying to prove. A good place for us to start would be with the definition of a metric space.

Definition (Metric space). Let X be a set and d : X × X →ℝ≥0 be a map (called the metric) with the following properties (for all x,y,z ∈ X):

  • d(x,y) = 0 ⇔ x = y;
  • d(x,y) = d(y,x) (symmetry);
  • d(x,y) ≤ d(x,z) + d(z,y) (triangle inequality).

We say that the pair (X,d) is a metric space. For x,y ∈ X, we will often call the value d(x,y) the distance between x and y.

Remark. While this definition of a metric space might be intuitive, it is not particularly accurate to the real world. For example, the property that for all x,y ∈ X that d(x,y) = d(y,x) is not true when driving in a city; one way roads make it so the route from point A to B could be different from the route from B to A, and hence the distance could also be different. There are variations of metric spaces which alter the properties, such as quasimetrics (which omits the symmetric property) and semimetrics (which omits the triangle inequality).

Next we define what we mean by the diameter of a metric space.

Definition (Diameter). Let (X,dX) be a metric space. Then we define the diameter of X to be diam(X) = sup{dX(x,y)|x,y ∈ X}. We say that X has finite diameter if diam(X) < ∞.

There are numerous ways of studying the similarity between metric spaces. One of the most strict notions of similarity is that of an isometry, where the distance between two elements in the domain must be equal to the distance of the images of those two elements in the range. We are interested in a more relaxed notion of similarity. Before we define it however, we need to understand what it means for two mappings to be a finite distance from each other.

Definition (Finite distance). Let (X,dX) and (Y,dY ) be metric spaces and f,f′ : X → Y be two mappings between them. We say that f is a finite distance from f′ if there exists a c ∈ℝ≥0 such that for all x ∈ X, dY (xf,xf′) ≤ c.

With this definition, we can now define what a quasi-isometry is.
